# ![openlink](./logo/openlink-full-path.svg)
<p>OpenLink is an open-source control and telemetry software system for Western Engineering's Trillium Hyperloop.</p>

<p>&nbsp;</p>

# Pod
## Requirements
- rust (> 1.56)

## How to run
From the pod directory:

```
cargo run
```

<p>&nbsp;</p>

# Remote Client
## Requirements
- node.js LTS (~16.13)
- rust (> 1.56)
- yarn

## How to install/run
From the remote directory:

```
yarn install
yarn tauri:serve
```
